BMW M5 Competition — Interactive Showcase Website
A dark-themed interactive car showcase website built with HTML, CSS and JavaScript.

Live Features
- Hero section with car image and full specifications
- Interactive tab switcher — Front, Rear, Top and Interior views
- Animated spec progress bars on each view
- Smooth fade-in scroll animations
- Nav bar that darkens on scroll
- Clickable feature tags

Tech Stack
- HTML5
- CSS3 (Grid, Flexbox, CSS Variables, Animations)
- Vanilla JavaScript (IntersectionObserver, classList API)

Project Structure

bmw-site/
├── index.html
├── style.css
├── script.js
└── images/
      ├── front.jpeg
      ├── rear.jpeg
      ├── top.jpeg
      └── interior.jpeg

How to Run
1. Clone or download this repository
2. Open index.html with Live Server in VS Code
3. That's it — no dependencies or installs needed
